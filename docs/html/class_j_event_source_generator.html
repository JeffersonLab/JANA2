Detailed Description

This is a base class for all event source generators.

JANA implements event sources in a modular way so that new types of sources can be easily added. Typically, this just means different file formats, but can also be networked or shared memory sources.

One may subclass the JEventSource class directly, but it is recommended to use the template class JEventSourceGeneratorT instead. That defines defaults for the required methods based on the JEventSource class the generator is for while still allowing to customization where needed. See the JEventSourceGeneratorT documentation for details.
